Key Features

🛡

6-Layer Spam & Malware Detection

Trained on 1,645 real spam samples with 82,000+ detection tokens. Bayesian analysis, domain validation, header authentication, heuristic pattern matching, OS antivirus integration, and YARA malware scanning work together to stop spam and malware before you see it. Detects trojans, ransomware, malicious macros, and phishing documents.

⚡

Quick Rules — Intelligent Triage

Create one-click rules to quarantine, move, or auto-delete email from specific senders. Rules grow smarter as you use them — apply to one sender and it’s added to the match list automatically. Scan entire folders in seconds.

📨

Large Mailbox Support

Your newest 500 messages load in under a minute. The rest syncs quietly in the background while you work — even mailboxes with 100,000+ messages. Pause, resume, or stop anytime.

💾

Automatic Encrypted Backups

Full weekly backups and lightweight daily differentials, all AES-256 encrypted. Every deleted message is captured for 90 days. Browse any backup instantly and recover individual messages with one click.

👁

External Image & Tracking Pixel Protection

External images blocked by default to prevent IP tracking. Tracking pixels stripped to prevent read receipts. Independent controls — load images without loading trackers.

💻

Local-First. No Cloud.

Every email stored on YOUR computer in an encrypted SQLite database. No accounts to create. No subscriptions. Community spam intelligence sharing is strictly opt-in and off by default — only anonymized filter scores, never email content.

🌐

Five Platforms. One Experience.

Windows (code signed), macOS (Apple Silicon), Linux (x64), iOS, and Android. Same privacy-first approach on every platform. Desktop versions are full-featured; the mobile app brings your local inbox to your pocket.

🧠

Filter That Learns

Every time you mark a message as spam or safe, the filter gets smarter. It starts with 82,000+ pre-trained patterns and builds on them as you use it. Learn more →
